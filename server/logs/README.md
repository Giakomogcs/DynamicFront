# Logs Directory

This directory contains application logs generated by Winston.

## Log Files

- `auth.log` - Authentication-related events
- `combined.log` - All application logs
- `error.log` - Error logs only

## Structure

All logs are in JSON format for easy parsing and analysis.

Example log entry:
```json
{
  "timestamp": "2026-01-20 12:00:00",
  "level": "info",
  "message": "Profiles loaded",
  "event": "PROFILE_LOAD",
  "resourceId": "55552e89-e63d-4362-b061-1077b8289eef",
  "profileCount": 3
}
```

## Retention

- Maximum file size: 5MB
- Maximum files kept: 5 per type
- Older logs are automatically rotated
